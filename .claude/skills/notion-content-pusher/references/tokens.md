> ⚠️ **CẢNH BÁO:** File này chứa token truy cập Notion.
> - **KHÔNG commit** lên GitHub
> - **KHÔNG chia sẻ** file này với bất kỳ ai
> - Lưu trong `.gitignore` nếu dùng Git

## Hướng dẫn thêm token mới

Khi thêm token mới, thêm theo format:

```markdown
## [Tên hiển thị]
- Display name: [Tên dễ nhớ]
- Token: ntn_xxxxxxxxxxxxxxxxxxxx
- Workspace ID: [Lấy từ API response]
- Ngày thêm: YYYY-MM-DD
```
