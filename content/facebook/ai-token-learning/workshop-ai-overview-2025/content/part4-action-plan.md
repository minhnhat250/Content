## MODULE 2 — 7 NƯỚC ĐI CỤ THỂ

### NƯỚC ĐI #1 — Bắt Đầu Dùng AI Như "Cộng Sự"

> **Mục tiêu:** Biến AI từ "chat toy" → "cộng sự làm việc"

**Checklist tuần này:**

- [ ] Ngày 1–2: Cài đặt Claude/ChatGPT trên điện thoại + desktop
- [ ] Ngày 3: Dùng AI cho 1 task NHÀM CHÁNG (viết email, tóm tắt document, sửa bài viết)
- [ ] Ngày 4–5: Thử "nâng cao" — đặt câu hỏi phức tạp hơn
- [ ] Ngày 6–7: Viết RA 3 prompt bạn sẽ dùng MỖI NGÀY

**Prompt Templates để bắt đầu:**

| Task | Template |
|---|---|
| **Email** | "Viết email cho [tên] về [chủ đề], tone [chuyên nghiệp/thân mật], khoảng [X] từ" |
| **Tóm tắt** | "Tóm tắt đoạn text sau thành [X] điểm chính: [text]" |
| **Brainstorm** | "Brainstorm [X] ý tưởng cho [vấn đề]. Mỗi ý có pros/cons." |
| **Review** | "[Dán code hoặc logic]. Tìm bugs và đề xuất cải thiện." |

> **ĐIỂM ĂN TIỀN:**
> Nhân viên dùng AI tốt = tiết kiệm 2–4h/ngày = tăng productivity 30–50%.
> → Thăng tiến nhanh hơn, làm được nhiều việc hơn với ít effort hơn.

---

### NƯỚC ĐI #2 — Học "Prompting" — Kỹ Năng 2026

> **Mục tiêu:** Từ "hỏi lục cục" → "ra lệnh chính xác"

**5 Nguyên Tắc Prompting Hiệu Quả:**

**1. BE SPECIFIC — Rõ ràng nhất có thể**
- ❌ "Viết bài về AI"
- ✅ "Viết bài 500 từ về tác động của AI lên ngành marketing, cho audience là marketer Việt Nam, tone chuyên nghiệp"

**2. CHOOSE ROLE — Gán vai trò cho AI**
- ❌ "Viết job description"
- ✅ "Bạn là Senior HR Manager với 10 năm kinh nghiệm. Viết job description cho vị trí Marketing Manager"

**3. GIVE CONTEXT — Cung cấp ngữ cảnh**
- ❌ "Phân tích data này"
- ✅ "Phân tích data bán hàng Q1 2025 cho công ty bán shoes online. Tìm 3 insights quan trọng nhất"

**4. ASK FOR FORMAT — Chỉ định format**
- ❌ "Liệt kê các bước"
- ✅ "Liệt kê 5 bước dưới dạng markdown checklist với emojis"

**5. ITERATE — Hỏi đi hỏi lại**
- Câu trả lời đầu → thường chưa tốt nhất
- "Hay hơn / Chi tiết hơn / Tối ưu hơn..."

**Challenge tuần 2:**
> Dùng 5 nguyên tắc trên → viết 10 prompts cho công việc CỤ THỂ của bạn.
> → Lưu vào Notion/Google Docs → Dùng hàng ngày.

> **ĐIỂM ĂN TIỀN:**
> Prompt Engineer đang là role HOT. Freelance rate: $50–150/giờ.
> → Biết prompting tốt = freelancer AI content/prompts.

### NƯỚC ĐI #3 — Xây Dựng "AI Workflow"

> **Mục tiêu:** Không dùng AI từng lần → Mà xây HỆ THỐNG tự động

**Ví dụ AI Workflow theo ngành:**

| Ngành | Workflow |
|---|---|
| **Marketer** | Research AI (Perplexity) → Content Draft AI (Claude) → Review (Human) → Publish (Buffer) |
| **Developer** | Bug Report AI (Claude) → Code Fix AI (Claude) → Test (Human) → Deploy (CI/CD) |
| **Data Analyst** | Data Clean AI (Claude/Python) → EDA AI (Claude) → Visualization (Python) → Report (Canva) |
| **Sales** | Lead Research AI (Perplexity) → Email Draft AI (Claude) → Personalize (Human) → Send (Outreach) |

**Công cụ cần học:**

| Level | Công cụ | Độ khó |
|---|---|---|
| Cơ bản | ChatGPT/Claude + Zapier | ⭐ |
| Trung bình | Make.com, n8n | ⭐⭐ |
| Nâng cao | Python + API | ⭐⭐⭐ |

> **ĐIỂM ĂN TIỀN:**
> Xây AI workflow cho doanh nghiệp = consulting fee $5K–30K/project.
> → Một freelancer xây 2–3 cái/month = $10K–90K/tháng.

### NƯỚC ĐI #4 — Học Python

> **Mục tiêu:** Đọc code AI, tự build đơn giản, communicate với developers

**Lộ trình Python cho người KHÔNG muốn trở thành developer:**

**Tháng 1–2: Python Cơ Bản**

- Biến, vòng lặp, hàm, list, dictionary
- Đủ để đọc code + viết script đơn giản
- Công cụ: Kaggle Python Course (FREE)

**Tháng 3–4: Data với Python**

- Pandas (đọc data, phân tích)
- NumPy (tính toán)
- Đủ để phân tích data như Excel nhưng MẠNH HƠN
- Công cụ: Kaggle Pandas Course (FREE)

**Tháng 5–6: ML/DL Introduction**

- scikit-learn (ML cơ bản)
- PyTorch/TensorFlow basics
- Hugging Face (dùng LLM)
- Đủ để build AI app đơn giản

> **ĐIỂM ĂN TIỀN:**
> Biết Python + AI = Data Analyst/Scientist = $60–120K/năm.
> → Freelancer Python developer = $50–100/giờ.

## MODULE 3 — 5 CAREER PATHS + THU NHẬP THỰC TẾ 2026 🔬VERIFY

---

**PATH 1 — AI POWER USER** *(Non-Tech)*

- Dùng AI tools + workflow để TĂNG NĂNG SUẤT
- Không cần code nhiều
- **Thu nhập:** Tăng productivity → Thăng tiến trong ngành
- **Thời gian:** 1–3 tháng
- **Phù hợp:** Văn phòng, marketing, sales, operations

---

**PATH 2 — AI PROMPT ENGINEER / CONSULTANT**

- Thiết kế AI workflow, tối ưu prompts cho doanh nghiệp
- **Freelance:** $50–150/giờ
- **Full-time:** $60–100K/năm
- **Thời gian:** 3–6 tháng
- **Phù hợp:** Marketer, content, operations (biết một chút tech)

---

**PATH 3 — AI BUILDER / FREELANCER**

- Build AI app, chatbot, RAG system, automation cho client
- **Freelance:** $5K–30K/project
- **Full-time AI Engineer:** $80–150K/năm 🔬VERIFY
- **Thời gian:** 6–12 tháng (học Python + AI)
- **Phù hợp:** Developers, students, career switchers

---

**PATH 4 — ML/DL ENGINEER** *(Specialist)*

- Xây dựng, train, optimize ML/DL models
- **Full-time:** $100–200K/năm 🔬VERIFY
- **Thời gian:** 1–2 năm (học sâu)
- **Phù hợp:** Developers muốn chuyên sâu AI

---

**PATH 5 — AI PRODUCT MANAGER / STRATEGIST**

- Kết hợp business + AI → Định hướng AI product/strategy
- **Full-time:** $120–200K/năm 🔬VERIFY
- **Thời gian:** 1–2 năm (nếu có background business/tech)
- **Phù hợp:** Founders, product managers, business analysts
